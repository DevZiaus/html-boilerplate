# This is a HTML boilerplate with

    + Dark Mode
    + Light Mode
    + Responsive navbar

## Stack

    + HTML
    + CSS
    + Vanilla JavaScript

### Usage

Clone the repository
`git clone https://github.com/DevZiaus/html-boilerplate.git`

Go into to directory
`cd html-boilerplate`

Change the color pallete in `/assets/css/styles.css` and

Change the logo in `/assets/images/logo.png`
